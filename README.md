📝 TaskFlow Pro – Week 2 Internship Task

📌 Overview
Developed as part of the Flutter Development Internship (Week 2). This project demonstrates **State Management** using `setState` and **Local Data Persistence** using `SharedPreferences`.

🎯 Learning Objectives Achieved
- ✅ **State Management:** Using `setState` to build a reactive UI.
- ✅ **Data Persistence:** Saving and retrieving data locally so it remains after app restarts.
- ✅ **Data Serialization:** Converting custom Objects/Lists to JSON strings.
- ✅ **Modern UI:** Implementing a dashboard with gradients and SliverAppBars.

✨ Features
- 🔢 **Persistent Counter:** Increase/Decrease a value that stays saved.
- 📋 **To-Do List:** Add and delete tasks.
- 💾 **Automatic Saving:** Data is saved instantly whenever a change occurs.
- 📱 **Professional UI:** Clean, responsive design for Android/iOS.

🛠 Technologies Used
- Flutter / Dart
- [SharedPreferences](https://pub.dev/packages/shared_preferences) (Local Storage)
- Material UI 3
